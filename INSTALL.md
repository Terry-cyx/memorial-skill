# 纪念.skill 安装与使用指南

> 面向普通用户，不需要编程基础。

---

## 你需要准备什么

| 项目 | 说明 |
|------|------|
| **一台电脑** | Windows 10/11，有 NVIDIA 显卡（GTX 1060 以上，推荐 RTX 3060+） |
| **微信 PC 版** | 需要安装 3.9.x 旧版本（下面会教你） |
| **手机微信** | 用来迁移聊天记录到电脑 |
| **Python 3.10+** | 如果没有，去 https://python.org 下载安装 |

---

## 第一步：导出微信语音（约 10 分钟）

这是**唯一需要你手动操作**的步骤。之后的一切都是自动的。

### 1.1 安装微信旧版本

目前的开源工具还不支持微信 4.x，需要装 3.9.x 版本。

1. 下载微信 3.9.9.43：
   - 地址：https://github.com/tom-snow/wechat-windows-versions/releases/tag/v3.9.9.43
   - 点击 `WeChatSetup-3.9.9.43.exe` 下载

2. 双击安装，安装位置随意

3. 打开微信 3.9.9.43，**扫码登录**

> 新旧微信可以共存，但同一时间只能登录一个。导完语音后可以切回新版。

### 1.2 把手机聊天记录迁移到电脑

在**手机微信**上操作：

1. 打开微信 → **设置** → **聊天记录管理**
2. 点 **导入与导出** → **导出到电脑**
3. 选择 **仅导出指定聊天记录**
4. 选择你要导出的**群聊或个人对话**（比如家庭群）
5. 选择**时间范围**（建议选"全部"，语音越多越好）
6. 确认导出，等待完成

> 手机和电脑需要连同一个 WiFi 网络。
> 导出时间取决于聊天记录的量，几百条语音大约 1-3 分钟。

### 1.3 确认导出成功

迁移完成后，电脑微信里应该能看到那个群聊的所有消息。
在群里播放一条语音消息，能听到声音就说明迁移成功。

## 第二步：告诉 AI 开始（全自动）

在 Claude Code 中输入：

```
/create-memorial
```

AI 会引导你完成以下流程（全部自动，你只需要回答几个问题）：

1. **采集信息**：告诉 AI 这个人的名字、关系、性格
2. **提取语音**：AI 自动解密微信数据库，找到目标人的语音
3. **预处理**：自动转格式、降噪
4. **训练声音**：自动训练语音模型（约 15-30 分钟）
5. **生成档案**：追忆档案 + 人格档案 + 声音模型

---

## 第三步：使用纪念档案

训练完成后，你可以：

### 文字追忆
```
> 给我讲讲爷爷年轻时候的事
> 如果告诉他我考上大学了，他会说什么？
> 帮我写一篇关于他的悼词
```

### 听到 ta 的声音
```
> 用爷爷的声音说"吃亏是福"
> 用爷爷的声音读这段话
```

### 继续补充
```
> 我又找到了一些照片
> 我想补充一个故事
> 他不会这样说，他其实是...
```

## 常见问题

### Q: 我没有 NVIDIA 显卡能用吗？
A: 文字档案功能完全可以用。声音训练需要 NVIDIA 显卡（至少 4GB 显存），没有显卡的话可以用 CosyVoice 的零样本模式（效果稍差但不需要训练）。

### Q: 微信语音太少怎么办？
A: 30 秒以上就能产出可辨认的声音。3-5 分钟效果良好，10 分钟以上效果极佳。如果语音太少，系统会自动切换到零样本模式。

### Q: 可以给多个亲人建档案吗？
A: 可以，每个人独立一份，互不影响。

### Q: 数据安全吗？
A: 所有数据只在你的电脑上，不会上传任何服务器。

### Q: 迁移聊天记录后可以切回新版微信吗？
A: 可以。迁移完成后，关掉 3.9.x，打开 4.x 重新登录即可。数据已经在电脑上了。

### Q: Android 和 iPhone 都支持吗？
A: 都支持。迁移聊天记录的功能是微信自带的，Android 和 iPhone 操作步骤相同。
